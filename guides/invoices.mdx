## Render it

<CodeGroup>
```bash cURL
curl -s https://api.renderbase.app/v1/render \
  -H "Content-Type: application/json" \
  -H "x-api-key: pk_live_YOUR_KEY" \
  -d '{"html": "PASTE_MINIFIED_HTML_HERE"}' | jq .
```

```javascript Node.js
const html = `<!-- paste the template above -->`;

const res = await fetch("https://api.renderbase.app/v1/render", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "x-api-key": "pk_live_YOUR_KEY",
  },
  body: JSON.stringify({ html }),
});

const { url } = await res.json();
console.log("Download:", url);
```

```python Python
import requests

html = """<!-- paste the template above -->"""

res = requests.post(
    "https://api.renderbase.app/v1/render",
    headers={
        "Content-Type": "application/json",
        "x-api-key": "pk_live_YOUR_KEY",
    },
    json={"html": html},
)

print("Download:", res.json()["url"])
```
</CodeGroup>

## Tips

- **Use inline CSS.** External stylesheets work but add latency. Inline styles render fastest.
- **Google Fonts work.** The `@import url(...)` pattern loads web fonts during rendering.
- **Use `printBackground: true`** (the default) to keep background colors in the PDF.
- **Test your HTML locally first.** Open it in a browser to preview before sending to the API.
- **Keep HTML under 2MB.** For large documents, consider splitting into multiple pages.
